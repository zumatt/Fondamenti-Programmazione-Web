Che cosa sono gli elementi audio?

Gli elementi audio sono elementi multimediali che permettono di riprodurre file audio all'interno di una pagina web

Per inserire un file audio si utilizza il tag audio con l'attributo src che indica il percorso del file audio

Per indicare un testo alternativo al file audio si utilizza l'attributo alt
