Che cosa sono gli elementi unordered lists e ordered lists?

Le liste sono elementi che permettono di elencare una serie di elementi in modo ordinato o non ordinato

Le liste non ordinate si creano con il tag ul e ogni elemento della lista si crea con il tag li

  • Elemento 1
  • Elemento 2
  • Elemento 3

Le liste ordinate si creano con il tag ol e ogni elemento della lista si crea con il tag li

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
