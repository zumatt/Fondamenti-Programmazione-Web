In questa sezione possiamo vedere il Box Model, un meccanismo di layout the i browser utilizzano per renderizzare il contenuto organizzato in elementi.

Il modello si compone di 5 dichiarazioni differenti:

  • Larghezza o Width
  • Altezza o Height
  • Padding
  • Bordi o Border
  • Margini o Margin
